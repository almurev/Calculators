# Проект "Калькуляторы"

## Используемые технологии
* Python 3.12
* PyQt6

## Структура проекта
* main.py — основной файл калькуляторов
* qt_calculators_all.ui/ui_calculators.py - дизайн калькуляторов

## Установка и запуск
1. Клонировать репозиторий:
   git clone <ссылка_на_репозиторий> далее ссылка: https://github.com/almurev/Calculators
2. Создать виртуальное окружение:
   python -m venv .venv
3. Активировать окружение:
   .venv\Scripts\activate
4. Установить зависимости:
   pip install pyqt6
5. Проверить установленные зависимости:
   pip list
6. Запустить приложение:
   python main.py
